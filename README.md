# 🛍️ Amazon Reviews Sentiment Analysis using NLP & Machine Learning

## 📌 Project Overview
This project builds a Sentiment Analysis system using Natural Language Processing (NLP) techniques to classify Amazon product reviews into:

- Positive 😊  
- Negative 😡  
- Neutral 😐  

The system helps businesses analyze customer feedback and improve product quality.
